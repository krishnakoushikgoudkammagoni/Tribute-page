# Tribute Page – Rohit Sharma

## 📌 Project Description

This project is a simple Tribute Page created using HTML and CSS.
The webpage is dedicated to **Rohit Sharma**, one of the greatest Indian cricketers and the captain of the Indian national cricket team in limited-overs formats.

The page displays basic information about Rohit Sharma, including a quote, an image, and a short biography. It is designed to practice basic web development concepts such as page structure, styling, and layout.

## 🛠️ Technologies Used

* HTML5
* CSS3

## ✨ Features

* Header section with title and tagline
* Inspirational quote about hard work
* Image of Rohit Sharma
* Short biography section
* Footer with author information
* Simple and clean webpage layout

## 📖 About the Tribute

Rohit Sharma, also known as *The Hitman*, is one of the most successful batsmen in international cricket. He is known for his elegant batting style and holds the record for the highest individual score in One Day Internationals (264).
